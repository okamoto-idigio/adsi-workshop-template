# 画面設計

## 画面一覧

| # | 画面名 | パス | ロール | 説明 |
|---|--------|------|--------|------|
| 1 | ログイン | `/login` | 全員 | ユーザー名を入力してログイン |
| 2 | 打刻 | `/` | 全員 | 出勤・退勤ボタン + 当日の状態表示 |
| 3 | 勤怠一覧（自分） | `/attendance` | 全員 | 月選択 + 日別勤怠テーブル |
| 4 | 勤怠一覧（管理者） | `/admin/attendance` | 管理者 | 月選択 + 全社員の勤怠テーブル |

## ワイヤーフレーム

### 1. ログイン画面 (`/login`)

```
┌─────────────────────────────────────────┐
│           勤怠管理システム                 │
│                                         │
│  ┌─────────────────────────────┐        │
│  │ ユーザー名                    │        │
│  └─────────────────────────────┘        │
│                                         │
│         [ ログイン ]                      │
│                                         │
│  ※ エラー時: 「ユーザーが見つかりません」   │
└─────────────────────────────────────────┘
```

### 2. 打刻画面 (`/`) — ログイン後のホーム

```
┌─────────────────────────────────────────┐
│ [打刻] [勤怠一覧] [管理者一覧*]   田中花子 │
├─────────────────────────────────────────┤
│                                         │
│         2026年7月14日（月）               │
│                                         │
│  ┌─────────┐     ┌─────────┐           │
│  │  出勤   │     │  退勤   │           │
│  └─────────┘     └─────────┘           │
│                                         │
│  状態: 出勤済み（09:00）                  │
│                                         │
└─────────────────────────────────────────┘
* 管理者のみ表示
```

**ボタン状態**:
- 未出勤: 出勤ボタン有効、退勤ボタン無効
- 出勤済み: 出勤ボタン無効、退勤ボタン有効
- 退勤済み: 両方無効

## ナビゲーション

```
ログイン → 打刻画面（ホーム）
              │
              ├── 勤怠一覧（自分）
              └── 管理者一覧（ADMINのみ）
```

## コンポーネント構成（Next.js）

```
app/
├── login/
│   └── page.tsx          -- ログイン画面
├── page.tsx              -- 打刻画面（ホーム）
├── attendance/
│   └── page.tsx          -- 勤怠一覧（自分）
├── admin/
│   └── attendance/
│       └── page.tsx      -- 管理者一覧
└── components/
    ├── Header.tsx        -- ナビゲーションヘッダー
    ├── ClockButton.tsx      -- 出退勤ボタン
    ├── AttendanceTable.tsx  -- 勤怠テーブル（共通）
    ├── MonthSelector.tsx    -- 月選択 UI
    └── EmployeeSelector.tsx -- 社員選択ドロップダウン（管理者画面）
```
