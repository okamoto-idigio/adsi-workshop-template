# 勤怠管理アプリ — 要求仕様書（MVP）

## 概要

社員が Web ブラウザから出退勤を打刻し、自分の勤怠履歴を月次で確認できるアプリケーション。
管理者は全社員の勤怠一覧を閲覧できる。

---

## ユーザーストーリー

### 社員（一般ユーザー）

| ID | ストーリー | 受け入れ基準 |
|----|-----------|-------------|
| US-01 | 社員として、ユーザー名を入力してログインしたい | ユーザー名のみで認証（パスワード不要）。未登録のユーザー名はエラー |
| US-02 | 社員として、出勤ボタンを押して出勤時刻を記録したい | 現在時刻が出勤時刻として保存される。同日に2回押すとエラー |
| US-03 | 社員として、退勤ボタンを押して退勤時刻を記録したい | 現在時刻が退勤時刻として保存される。出勤前に押すとエラー。同日に2回押すとエラー |
| US-04 | 社員として、自分の月次勤怠一覧を確認したい | 指定月の日別に出勤時刻・退勤時刻・勤務時間が一覧表示される |

### 管理者

| ID | ストーリー | 受け入れ基準 |
|----|-----------|-------------|
| US-05 | 管理者として、全社員の月次勤怠一覧を確認したい | 全社員の指定月の勤怠が一覧表示される |

---

## 機能要件

### F-01: ログイン

- ユーザー名のみで認証する（パスワードなし）
- 登録済みユーザーのみログイン可能（マスタデータ）
- ログイン後、ロール（一般社員 / 管理者）に応じた画面を表示する

### F-02: 打刻（出勤・退勤）

- 出勤ボタン: 現在時刻を出勤時刻として記録する
- 退勤ボタン: 現在時刻を退勤時刻として記録する
- バリデーション:
  - 同日に出勤を2回打刻 → エラー
  - 出勤していない状態で退勤 → エラー
  - 同日に退勤を2回打刻 → エラー
- 打刻方法: Web ブラウザのボタン操作のみ
- 1日1回の出退勤（中抜けなし）
- 打刻忘れの修正機能なし（MVP スコープ外）

### F-03: 勤怠一覧（自分）

- 月を選択して自分の勤怠を日別に表示する
- 表示項目: 日付、出勤時刻、退勤時刻、勤務時間
- 勤務時間の計算ルール:
  - 勤務時間 = 退勤時刻 − 出勤時刻 − 休憩時間
  - 休憩: 12:00〜13:00（1時間）を自動控除
  - 12:00〜13:00 をまたがない勤務の場合は控除なし（0分）

### F-04: 勤怠一覧（管理者・全社員）

- 管理者ロールのユーザーのみアクセス可能
- 全社員の指定月の勤怠を一覧表示する
- 表示項目: 社員名、日付、出勤時刻、退勤時刻、勤務時間

## 勤務時間の計算ルール（詳細）

```
実労働時間 = 退勤時刻 − 出勤時刻 − 休憩控除時間

休憩控除時間:
  - 出勤時刻 < 13:00 かつ 退勤時刻 > 12:00 → 60分控除
  - それ以外 → 0分
```

### 計算例

| 出勤 | 退勤 | 休憩控除 | 勤務時間 |
|------|------|---------|---------|
| 9:00 | 18:00 | 60分 | 8時間00分 |
| 9:00 | 12:00 | 0分 | 3時間00分 |
| 13:00 | 18:00 | 0分 | 5時間00分 |
| 10:00 | 15:00 | 60分 | 4時間00分 |
